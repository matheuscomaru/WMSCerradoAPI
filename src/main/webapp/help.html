Documentação da API

Buscar produto por ID (codigo)

Método: GET

Endpoint: 179.190.0.181:8093/WMSCerradoApi/produto

Descrição: Retorna as informações de um produto com base no código fornecido.

Parâmetros
Parâmetro Tipo Descrição Obrigatório
id Integer Informar o código do produto no CH ERP. Sim
Exemplo de Requisição
GET 179.190.0.181:8093/WMSCerradoApi/produto?id=103
Exemplo de Resposta
{
  "productCode": "000103",
  "description": "NINO DOG ADULTO CARNE 06 KG",
  "ean": "7898674131990",
  "dun": "",
  "packingInfo": "",
  "factoryCode": "103",
  "dunQuantity": 120,
  "grossWeight": 6.06,
  "netWeight": 6,
  "palletDataQuantity": 120
}

Listar produtos

Método: GET

Endpoint: 179.190.0.181:8093/WMSCerradoApi/listarprodutos

Descrição: Retorna uma lista de produtos. Nessa lista será retornada somente produtos ativos, com tipo Revenda ou Acabado e que tenha código de barras informado no cadastro.

]

Listar OPs (Ordens de Produção)

Método: GET

Endpoint: 179.190.0.181:8093/WMSCerradoApi/listarops

Descrição: Retorna uma lista de ordens de produção lançadas no sistema.

Parâmetros
Parâmetro Tipo Descrição Obrigatório
datainicial String Data incial de lançamento a ser considerada no padrão yyyy-MM-dd Sim
datafinal String Data final de lançamento a ser considerada no padrão yyyy-MM-dd Sim
situacao Integer Código da situação da OP sendo 0:aberto 1:finalizada 2:estornada Sim
Exemplo de Requisição
GET 179.190.0.181:8093/WMSCerradoApi/produto?id=103
Exemplo de Resposta
[
    {
        "id": "001129",
        "productionBatch": "0026/11-2024",
        "expirationDate": "2025-11-27",
        "manufacturingDate": "2024-11-28",
        "note": "",
        "items": [
            {
                "productId": "000107",
                "quantity": 6000
            }
        ]
    }
]
